# 🛒 Birmarket Clone

A simple e-commerce catalog project built with **HTML, CSS and Vanilla JavaScript**.

## Features

- Dynamic categories
- Subcategory navigation
- Product listing
- SPA-like navigation using History API

## Technologies

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Fetch API
